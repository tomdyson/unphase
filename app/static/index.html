Audio Engineering Tools — No. 1

unphase

Drop two recordings of the same moment — a stereo pair and a spot mic, say — and the tool returns the sample-accurate delay and polarity needed to align them. It works out which file is the close mic from the correlation lag, so you don’t need to label them.

Load both inputs to continue Ready Result below

01 / What it does

Computes the cross-correlation of the two signals via FFT. The lag at peak correlation is the time offset; a negative peak means polarity is inverted.

02 / How to use it

Bounce 10–30 seconds of both tracks from your DAW — any portion with clear transients works best. The tool tells you which file is the close mic, so labelling isn’t required.

03 / Verify by ear

Apply the delay in a Sample Delay plugin, then null-test: flip polarity, solo both mics, and listen for maximum cancellation. That confirms the alignment is genuinely locked in.
